一个开箱即用，快速搭建大型应用的 Vue3 + Vite4.0 + TypeScript+...模板框架。集成了各类插件，并进行了模块化和按需加载的优化，可以放心使用。

# 快速开始

推荐使用 pnpm

```shell
pnpm install
```

运行命令

```shell
pnpm run dev
```
